# 🗡️ 一、战斗动作（Combat Actions）

## 1.1 基础攻击

| 动作名称 | 说明 | 备注 |
|---|---|---|
| 单手轻攻击（1H R1） | 单手持武器普通攻击 | 不同武器种类动作不同 |
| 单手重攻击（1H R2） | 单手持武器蓄力/强攻击 | 可蓄力 |
| 双手轻攻击（2H R1） | 双手持武器普通攻击 | 切换双持后解锁 |
| 双手重攻击（2H R2） | 双手持武器蓄力/强攻击 | 可蓄力 |
| 副手轻攻击（LH R1/L1） | 副手武器普通攻击 | 部分副手武器可独立攻击 |
| 蓄力重攻击（Charged R2） | 长按R2释放更大伤害 | 1H/2H 各有版本 |

## 1.2 派生攻击 / 特殊攻击

| 动作名称 | 说明 | 备注 |
|---|---|---|
| 翻滚攻击（Rolling Attack） | 翻滚后接R1 | 1H/2H各有不同动画 |
| 后撤步攻击（Backstep Attack） | 后撤步后接R1 | 1H/2H各有不同动画 |
| 冲刺攻击（Running/Dash Attack） | 奔跑中按R1 | 1H/2H各有不同动画 |
| 跳劈攻击（Jump Attack） | 前推+R2 | 破盾效果 |
| 踢击（Kick） | 前推+R1 | 部分武器为其他动作替代 |
| 下落攻击/跳落攻击（Plunging Attack） | 从高处坠落时按R1/R2 | 高伤害，常用于Boss战 |

## 1.3 连段/Combo

| 动作名称 | 说明 | 备注 |
|---|---|---|
| 轻攻击连段（R1 Combo） | R1→R1→R1→… | 每种武器链数和动画不同(2~6段不等) |
| 重攻击连段（R2 Combo） | R2→R2 | 部分武器支持 |
| 轻转重连段（R1→R2） | 轻攻击中途切换重攻击 | 部分武器有独特派生 |
| 重转轻连段（R2→R1） | 重攻击中途切换轻攻击 | 较少见 |

## 1.4 防御动作

| 动作名称 | 说明 | 备注 |
|---|---|---|
| 举盾格挡（Block/Guard） | L1持盾格挡 | 左手盾/武器均可 |
| 双手格挡（2H Block） | 双手持武器格挡 | 不同武器削减率不同 |
| 招架/弹反（Parry） | L2/小盾/匕首/部分武器 | 成功后敌人出现硬直 |

## 1.5 处决/致命一击

| 动作名称 | 说明 | 备注 |
|---|---|---|
| 背刺（Backstab） | 绕到敌人背后按R1 | 有独特处决动画 |
| 弹反处决/致命一击（Riposte） | 招架成功后按R1 | 有独特处决动画 |
| 破防处决（Guard Break Riposte） | 敌人护卫崩溃后按R1 | 动画同Riposte |
| 下落致命（Plunge Critical） | 对倒地/特定Boss的高落攻击 | 少数Boss专属机会 |

## 1.6 战技（Weapon Art / Skill）

| 动作名称 | 说明 | 备注 |
|---|---|---|
| 战技发动（L2 / 双持L2） | 每把武器拥有独特战技 | 消耗FP（蓝条） |
| 战技→派生攻击（Skill→R1/R2） | 部分战技有后续分支 | 如"架势→上挑/突刺" |
| 战技持续（Hold Skill） | 部分战技需要持续按键 | 如"坚守" |
| 双持武器战技（Paired Weapon L1） | 双持专属武器的L1特殊攻击 | 如双弯刀旋风斩 |

> **注意**：DS3中武器种类约 20+ 大类，每类有独立的 1H/2H 攻击动画组，具体包括直剑、大剑、特大剑、曲剑、大曲剑、刺剑、匕首、拳套/爪、鞭、斧、大斧、锤、大锤、枪/矛、戟、镰刀、弓/大弓/弩、法杖/圣铃/火焰等，每种的R1/R2/Rolling/Backstep/Running Attack/Combo动画均不同。

# 📊 总览统计

| 大类 | 子分类数 | 预估动作条目数 |
|---|---|---|
| **战斗动作** | 9个子类 | ~50+ 基础动作（×武器种类后成倍增加） |
| **交互动作** | 5个子类 | ~35+ 条 |
| **Locomotion** | 5个子类 | ~30+ 条 |
| **合计（不含手势）** | — | **~115+ 基础动作** |

> ⚠️ **特别说明**：
> - 上表列出的是**动作类型/行为种类**，实际动画文件数远远超过此数，因为每种武器类型（约20+大类）都有自己的一套攻击动画（1H/2H × R1/R2/Rolling/Backstep/Running/Combo/WeaponArt）。
> - 如果按武器种类展开，仅战斗攻击类动画就可达 **500+** 条。
> - 受击动画也有多方向（前/后/左/右）和多强度（轻/中/重）的排列组合。

## ?? **扇形锁定系统特性**

### **主要功能**
1. **扇形优先锁定区域**：前方100度（±50度）范围内的敌人优先被锁定
2. **边缘检测区域**：140度（±70度）范围内的敌人可以被锁定，但会触发相机修正
3. **智能相机修正**：锁定边缘目标时，相机会平滑旋转将目标带入主视野
4. **平滑过渡效果**：所有相机和角色转向都有平滑插值，体验流畅

### **系统工作原理**
1. **目标检测**：球体范围检测附近敌人
2. **区域分类**：将检测到的目标分为扇形区域和边缘区域
3. **优先级选择**：优先选择扇形区域内的目标
4. **自动修正**：边缘目标触发相机自动修正，将其带入主视野

### **技术实现**
- **扇形检测**：基于相机朝向的角度计算
- **相机修正**：异步定时器触发的平滑相机旋转
- **性能优化**：定时刷新候选目标，避免每帧计算

### **可调参数**
```cpp
SECTOR_LOCK_ANGLE = 100.0f      // 主锁定扇形角度
EDGE_DETECTION_ANGLE = 140.0f   // 边缘检测角度  
CAMERA_AUTO_CORRECTION_SPEED = 4.0f // 相机修正速度
CAMERA_CORRECTION_OFFSET = 30.0f    // 修正偏移角度
```

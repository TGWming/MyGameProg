迁移步骤 Checklist

□ 1. 复制整个模块文件夹 (SoulsCamera/)

□ 2.  修改 MainProject.uproject
     "Modules": [
         { "Name": "MainProject", ...  },
+        { "Name": "SoulsCamera", "Type": "Runtime", "LoadingPhase": "Default" }
     ]

□ 3. 修改 MainProject.Build.cs (如果角色代码需要引用)
+    PublicDependencyModuleNames.Add("SoulsCamera");

□ 4. 重新生成项目文件 (右键 . uproject → Generate Project Files)

□ 5. 编译

□ 6.  蓝图中添加组件 → 调参数 → 完成

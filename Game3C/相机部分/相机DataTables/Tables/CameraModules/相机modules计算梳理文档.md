┌─────────────────────────────────────────────────────────────────────────────┐
│  P01: FollowTarget (目标跟随)                                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  计算公式：                                                                  │
│  FocusPoint = CharacterLocation + FocusOffset                               │
│  或                                                                          │
│  FocusPoint = CharacterMesh->GetSocketLocation(FocusSocketName)             │
│                                                                              │
│  参数：                                                                      │
│  · FocusOffset: FVector(0, 0, 80) - 默认看向胸口                            │
│  · FocusSocketName: "spine_02" - 骨骼Socket名                               │
│  · bUseSocketOffset: false - 是否使用骨骼Socket                             │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  P02: FollowTarget_Lagged (惯性跟随)                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  计算公式（指数衰减）：                                                      │
│  Alpha = 1 - Exp(-LagSpeed × DeltaTime)                                     │
│  SmoothedPosition = Lerp(PreviousPosition, TargetPosition, Alpha)           │
│                                                                              │
│  特殊处理：水平和垂直分离惯性                                                │
│  · 水平惯性较快 (LagSpeed = 8. 0)                                            │
│  · 垂直惯性较慢 (VerticalLagSpeed = 4.0)                                    │
│                                                                              │
│  理论来源：知乎文章1 - 指数衰减插值公式                                      │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  P03: FollowTarget_Predictive (预测性跟随)                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  计算公式：                                                                  │
│  PredictedOffset = CharacterVelocity × PredictionTime                       │
│                                                                              │
│  参数：                                                                      │
│  · PredictionTime: 0. 1~0.2s                                                 │
│  · 速度阈值: 仅当速度 > 100 单位/秒时启用                                   │
│                                                                              │
│  理论来源：Mark Haigh-Hutchinson - Predictive Positioning                  │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  P04: Orbit_LockOn (锁定轨道)                                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  计算方式：                                                                  │
│  · 计算玩家到目标的方向向量                                                  │
│  · 相机位于玩家身后，面向目标                                                │
│  · 可选：焦点在玩家-目标中点的加权位置                                       │
│                                                                              │
│            Target                                                            │
│               ●                                                              │
│              /                                                               │
│             /   ← 视线方向                                                   │
│            /                                                                 │
│     Player ●─────────● Camera                                               │
│            ← OrbitRadius                                                    │
│                                                                              │
│  理论来源：Target-Relative 控制参考系                                        │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  P05: Orbit_Boss (Boss轨道)                                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  与P04的区别：                                                               │
│  · 根据Boss体型动态调整轨道半径                                              │
│  · 焦点可能偏向Boss身体中心而非顶部                                          │
│  · 巨型Boss时焦点下移（看向躯干而非头部）                                    │
│                                                                              │
│  参数映射：                                                                  │
│  · Medium Boss (300+): 基础轨道                                              │
│  · Large Boss (500+): 轨道×1.3 + 焦点下移                                   │
│  · Huge Boss (800+): 轨道×1.6 + 焦点大幅下移                                │
│                                                                              │

┌─────────────────────────────────────────────────────────────────────────────┐
│  R01: PlayerInput_Free (玩家自由输入)                                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  计算公式：                                                                  │
│  NewYaw = CurrentYaw + PlayerYawInput × Sensitivity × DeltaTime            │
│  NewPitch = CurrentPitch + PlayerPitchInput × Sensitivity × DeltaTime      │
│  NewPitch = Clamp(NewPitch, MinPitch, MaxPitch)                             │
│                                                                              │
│  参数：                                                                      │
│  · Sensitivity: 1.0 (可在设置中调整)                                        │
│  · MinPitch: -60°                                                           │
│  · MaxPitch: +70°                                                           │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  R04: LookAt_Target_Soft (软锁定看向)                                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  类魂游戏核心特性：锁定时仍允许玩家微调视角                                  │
│                                                                              │
│  计算公式：                                                                  │
│  BaseRotation = LookAt(TargetLocation)                                      │
│  PlayerOffset = PlayerInput × SoftLockStrength                              │
│  FinalRotation = BaseRotation + PlayerOffset                                │
│                                                                              │
│  参数：                                                                      │
│  · SoftLockStrength: 0. 3 (30%的玩家输入被响应)                              │
│  · SoftLockReturnSpeed: 5.0 (释放输入后回归速度)                            │
│                                                                              │
│  参考：血源诅咒/艾尔登法环的锁定手感                                         │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  R06: AutoOrient_Movement (移动方向自动朝向)                                │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  触发条件：角色高速移动时（冲刺/骑乘）                                       │
│                                                                              │
│  计算公式：                                                                  │
│  TargetYaw = CharacterVelocity. Rotation(). Yaw                               │
│  Alpha = 1 - Exp(-AutoOrientSpeed × DeltaTime)                              │
│  YawOffset = Lerp(0, TargetYaw - CurrentYaw, Alpha)                         │
│                                                                              │
│  参数：                                                                      │
│  · AutoOrientSpeed: 2.0                                                     │
│  · 速度阈值: 500 单位/秒                                                    │
│                                                                              │
│  理论来源：知乎文章1 - Input-Driven Recentering                             │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  R07: AutoOrient_Delayed (延迟自动回中)                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  触发条件：玩家长时间未输入视角控制                                          │
│                                                                              │
│  计算流程：                                                                  │
│  IF TimeSinceLastInput > AutoCorrectDelay                                   │
│      TargetYaw = CharacterRotation.Yaw + 180° (角色背后)                    │
│      Alpha = 1 - Exp(-AutoCorrectSpeed × DeltaTime)                         │
│      YawOffset = Lerp(0, TargetYaw - CurrentYaw, Alpha)                     │
│                                                                              │
│  参数：                                                                      │
│  · AutoCorrectDelay: 3.0s                                                   │
│  · AutoCorrectSpeed: 2.0                                                    │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘

│  F02: Speed_FOV (速度FOV)                                                   │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  计算公式（来自知乎文章1）：                                                 │
│  IF Speed > SpeedThreshold                                                  │
│      FOVOffset = (Speed - SpeedThreshold) × FOVPerSpeed                    │
│      FOVOffset = Clamp(FOVOffset, 0, MaxFOVOffset)                          │
│                                                                              │

│  · SpeedThreshold: 300 cm/s                                                 │
│  · FOVPerSpeed: 0.02 (每单位速度增加0.02度)                                 │
│  · MaxFOVOffset: 15° (最大增量)                                             │
│                                                                              │
│  平滑处理：                                                                  │
│  Alpha = 1 - Exp(-FOVLagSpeed × DeltaTime)                                  │
│  SmoothedFOVOffset = Lerp(Previous, Current, Alpha)                         │
│  · FOVLagSpeed: 5.0                                                         │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘

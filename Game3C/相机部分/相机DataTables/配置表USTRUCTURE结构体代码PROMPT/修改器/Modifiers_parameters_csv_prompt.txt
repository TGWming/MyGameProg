### 任务2：创建 CameraModifierParameterData. h
路径：Source/[YourProject]/Public/Camera/Data/CameraModifierParameterData.h

创建一个继承自 FTableRowBase 的 USTRUCT，用于从CSV导入相机修改器参数配置数据

CSV文件结构如下（共62行数据）：
- ParamID:  FString - 参数唯一标识符，格式如 "MS01", "MR01", "MC01", "MZ01", "ME01", "MX01"（注意：此字段会自动成为RowName，结构体中可保留但导入后为空）
- ParamName: FName - 参数名称，作为程序引用标识符（如 ShakeFrequency_Light, ParryDistancePull）
- ChineseName: FString - 中文显示名称
- Category: FString - 参数分类（Shake, Reaction, Cinematic, Zoom, Effect, Special）
- DataType: FString - 数据类型（float, FLinearColor）
- DefaultValue: FString - 默认值（数值如"25.0"，或颜色格式如"(1.0,0.2,0.2,1.0)"）
- MinValue: FString - 最小值（"-"表示无限制）
- MaxValue: FString - 最大值（"-"表示无限制）
- Unit: FString - 单位（如 "Hz", "cm", "degree", "s", "-"）
- Description: FString - 参数描述
- RelatedModifiers: FString - 关联修改器ID列表，用斜杠分隔（如 "S01", "R01", "C01/C02/C03"）

结构体命名为 `FCameraModifierParameterRow`

添加必要的UPROPERTY宏，确保：
- 所有属性可在编辑器中编辑（EditAnywhere）
- 所有属性可在蓝图中读取（BlueprintReadWrite）
- 添加Category分组便于编辑器查看

**重要：结构体中不要包含任何UFUNCTION函数，只保留UPROPERTY属性和构造函数**

### GetDefaultValueAsLinearColor
```cpp
FLinearColor Result = FLinearColor:: White;

if (Row.DefaultValue.IsEmpty() || Row.DefaultValue == TEXT("-"))
{
    return Result;
}

// 移除括号和引号
FString CleanedValue = Row.DefaultValue;
CleanedValue.RemoveFromStart(TEXT("\""));
CleanedValue.RemoveFromEnd(TEXT("\""));
CleanedValue.RemoveFromStart(TEXT("("));
CleanedValue.RemoveFromEnd(TEXT(")"));

// 按逗号分割
TArray<FString> Components;
CleanedValue.ParseIntoArray(Components, TEXT(","), true);

if (Components.Num() >= 4)
{
    Result. R = FCString:: Atof(*Components[0]. TrimStartAndEnd());
    Result.G = FCString:: Atof(*Components[1].TrimStartAndEnd());
    Result.B = FCString::Atof(*Components[2]. TrimStartAndEnd());
    Result.A = FCString:: Atof(*Components[3].TrimStartAndEnd());
}
else if (Components.Num() >= 3)
{
    Result.R = FCString:: Atof(*Components[0].TrimStartAndEnd());
    Result.G = FCString::Atof(*Components[1]. TrimStartAndEnd());
    Result.B = FCString:: Atof(*Components[2].TrimStartAndEnd());
    Result.A = 1.0f;
}

return Result;
```

## 特殊处理说明
- FLinearColor类型的DefaultValue格式为 "(r,g,b,a)"，如 "(1.0,0.2,0.2,1.0)"
- MinValue/MaxValue为"-"时表示无限制（FLinearColor类型始终为"-"）
- Unit字段仅供显示参考，不影响数值计算
- RelatedModifiers使用斜杠"/"分隔多个修改器ID（如 "C01/C02/C03"）
- ParamID格式说明：
  - MS开头：Shake类参数
  - MR开头：Reaction类参数
  - MC开头：Cinematic类参数
  - MZ开头：Zoom类参数
  - ME开头：Effect类参数
  - MX开头：Special类参数

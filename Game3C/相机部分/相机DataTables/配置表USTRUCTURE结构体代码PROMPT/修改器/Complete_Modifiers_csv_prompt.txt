### 任务2：创建 CameraModifierData.h
路径：Source/[YourProject]/Public/Camera/Data/CameraModifierData. h

创建一个继承自 FTableRowBase 的 USTRUCT，用于从CSV导入相机修改器配置数据

CSV文件结构如下（共27行数据）：
- ModifierID:  FString - 修改器唯一标识符，格式如 "CompleteModifiers_S01"（注意：此字段会自动成为RowName，结构体中可保留但导入后为空）
- ModifierName: FName - 修改器名称，作为程序引用标识符（如 Shake_Hit_Light, Reaction_Parry）
- ChineseName: FString - 中文显示名称
- Category: FString - 主分类（Shake, Reaction, Cinematic, Zoom, Effect, Special）
- SubCategory: FString - 子分类（Combat, Environment, Movement, Special, Boss, Status, Aim, Time, Death）
- TriggerEvent: FString - 触发事件描述（如 "玩家受到轻攻击", "弹反成功", "进入Boss区域"）
- Duration: FString - 持续时间（数值如"0.15"，或描述性文本如"动画长度", "持续", "模式持续", "慢动作持续", "状态持续", "3.0-5.0"）
- BlendInTime: float - 混合进入时间（秒）
- BlendOutTime: float - 混合退出时间（秒）
- Priority: int32 - 优先级（数值越大优先级越高，范围50-255）
- bAffectsPosition:  bool - 是否影响位置（注意：布尔类型字段命名以小写b开头）
- bAffectsRotation: bool - 是否影响旋转
- bAffectsFOV: bool - 是否影响FOV
- bAffectsPostProcess: bool - 是否影响后处理
- bIsCore: bool - 是否为核心修改器
- Description: FString - 修改器功能描述

结构体命名为 `FCameraModifierRow`

添加必要的UPROPERTY宏，确保：
- 所有属性可在编辑器中编辑（EditAnywhere）
- 所有属性可在蓝图中读取（BlueprintReadWrite）
- 添加Category分组便于编辑器查看

**重要：结构体中不要包含任何UFUNCTION函数，只保留UPROPERTY属性和构造函数**

## 函数实现逻辑参考

### GetDurationAsFloat
```cpp
if (Row.Duration. IsEmpty() || Row.Duration == TEXT("-"))
{
    return -1.0f;
}

// 检查是否为范围格式 "x.x-x.x"
if (Row.Duration.Contains(TEXT("-")) && Row.Duration.Left(1) != TEXT("-"))
{
    // 范围格式，返回-1表示需要用GetDurationRange获取
    return -1.0f;
}

// 尝试解析为数值
if (Row.Duration. IsNumeric() || 
    Row.Duration.Contains(TEXT(". ")) && 
    ! Row.Duration.Contains(TEXT("动画")) &&
    !Row.Duration.Contains(TEXT("持续")) &&
    !Row.Duration.Contains(TEXT("模式")))
{
    return FCString::Atof(*Row.Duration);
}

// 描述性文本，返回-1
return -1.0f;
```

### HasFixedDuration

### GetDurationRange
```cpp
OutMin = 0.0f;
OutMax = 0.0f;

if (Row.Duration. IsEmpty())
{
    return false;
}

// 检查是否为范围格式 "x.x-x.x"
int32 DashIndex = INDEX_NONE;
// 跳过开头可能的负号
FString CheckString = Row.Duration;
if (CheckString.StartsWith(TEXT("-")))
{
    return false; // 负数不是范围
}

if (CheckString.FindChar(TEXT('-'), DashIndex) && DashIndex > 0)
{
    FString MinStr = CheckString.Left(DashIndex).TrimStartAndEnd();
    FString MaxStr = CheckString.Mid(DashIndex + 1).TrimStartAndEnd();
    
    if (! MinStr.IsEmpty() && !MaxStr.IsEmpty())
    {
        OutMin = FCString:: Atof(*MinStr);
        OutMax = FCString:: Atof(*MaxStr);
        return OutMax > OutMin;
    }
}

return false;
```

## 特殊处理说明
- Duration字段有多种格式：
  - 固定数值：如 "0.15", "0.3", "2"
  - 范围值：如 "3.0-5.0", "2.0-3.0", "0.05-0.1"
  - 描述性文本：如 "动画长度", "持续", "模式持续", "慢动作持续", "状态持续"
- 布尔值在CSV中为 "true"/"false"

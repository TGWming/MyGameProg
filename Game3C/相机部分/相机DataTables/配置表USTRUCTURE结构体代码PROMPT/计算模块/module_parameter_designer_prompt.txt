### 任务2：创建 CameraModuleParameterData.h
路径：Source/[YourProject]/Public/Camera/Data/CameraModuleParameterData.h

创建一个继承自 FTableRowBase 的 USTRUCT，用于从CSV导入相机模块参数配置数据

CSV文件结构如下（共56行数据）：
- ParamID:  FString - 参数唯一标识符，格式如 "ModuleParameterDesigner_MP01"（注意：此字段会自动成为RowName，结构体中可保留但导入后为空）
- ParamName: FName - 参数名称，作为程序引用标识符（如 PositionLagSpeed, FocusOffset）
- ChineseName: FString - 中文显示名称
- Category: FString - 参数分类（Position, Rotation, Distance, FOV, Offset, Constraint）
- DataType: FString - 数据类型（float, FVector, FName）
- DefaultValue:  FString - 默认值（数值如"8"，或向量格式如"(0,0,80)"，或名称如"spine_02"）
- MinValue:  FString - 最小值（"-"表示无限制）
- MaxValue: FString - 最大值（"-"表示无限制）
- Unit: FString - 单位（如 "1/s", "cm", "degree", "s", "cm/s", "cm/(cm/s)", "degree/(cm/s)", "-"）
- Description: FString - 参数描述
- RelatedModules: FString - 关联模块ID列表，用斜杠分隔（如 "P01/P02" 或 "R04/R05"）

结构体命名为 `FCameraModuleParameterRow`

添加必要的UPROPERTY宏，确保：
- 所有属性可在编辑器中编辑（EditAnywhere）
- 所有属性可在蓝图中读取（BlueprintReadWrite）
- 添加Category分组便于编辑器查看

**重要：结构体中不要包含任何UFUNCTION函数，只保留UPROPERTY属性和构造函数**

### GetDefaultValueAsVector
```cpp
FVector Result = FVector::ZeroVector;

if (Row.DefaultValue. IsEmpty() || Row.DefaultValue == TEXT("-"))
{
    return Result;
}

// 移除括号和引号
FString CleanedValue = Row.DefaultValue;
CleanedValue.RemoveFromStart(TEXT("\""));
CleanedValue.RemoveFromEnd(TEXT("\""));
CleanedValue.RemoveFromStart(TEXT("("));
CleanedValue.RemoveFromEnd(TEXT(")"));

// 按逗号分割
TArray<FString> Components;
CleanedValue.ParseIntoArray(Components, TEXT(","), true);

if (Components.Num() >= 3)
{
    Result. X = FCString:: Atof(*Components[0]. TrimStartAndEnd());
    Result.Y = FCString:: Atof(*Components[1].TrimStartAndEnd());
    Result.Z = FCString::Atof(*Components[2].TrimStartAndEnd());
}

return Result;
```

## 特殊处理说明
- FVector类型的DefaultValue格式为 "(x,y,z)"，需要移除引号和括号后解析
- MinValue/MaxValue为"-"时表示无限制
- Unit字段仅供显示参考，不影响数值计算
- RelatedModules使用斜杠"/"分隔多个模块ID

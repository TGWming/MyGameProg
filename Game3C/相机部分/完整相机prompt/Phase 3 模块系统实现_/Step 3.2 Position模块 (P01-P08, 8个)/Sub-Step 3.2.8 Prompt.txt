### 4. 圆形轨道模拟

**数学原理：**
```
圆周运动公式：
X = CenterX + Radius * cos(Angle)
Y = CenterY + Radius * sin(Angle)

参数化：
Angle = Progress * 2π = Progress * 360°
Progress ∈ [0, 1]

实现：
float Angle = CurrentSplinePosition * 360.0f;
SplineOffset.X = cos(Angle_radians) * Radius;
SplineOffset.Y = sin(Angle_radians) * Radius;

效果：
- Progress=0.0 → 角色右侧（0°）
- Progress=0.25 → 角色前方（90°）
- Progress=0.5 → 角色左侧（180°）
- Progress=0.75 → 角色后方（270°）
- Progress=1.0 → 回到起点（360°=0°）
```

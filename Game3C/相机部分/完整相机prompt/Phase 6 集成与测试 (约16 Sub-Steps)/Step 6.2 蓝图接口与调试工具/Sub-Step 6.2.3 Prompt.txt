【SoulsCameraManager可用接口】
调试组件需要通过以下Manager接口获取数据：

状态查询：
- GetCurrentStateName() -> FName
- GetCurrentCategory() -> ECameraStateCategory
- IsInTransition() -> bool

目标查询：
- HasLockOnTarget() -> bool
- GetLockOnTarget() -> AActor*

输出查询（通过GetCurrentOutput()）：
- GetCurrentOutput() -> FSoulsCameraOutput
  - Output.FocusPoint (FVector)
  - Output. Rotation (FRotator)
  - Output. Distance (float)
  - Output.FOV (float)
  - Output.GetCameraLocation() -> FVector

碰撞查询：
- IsCollisionActive() -> bool
- IsInCollisionRecovery() -> bool
- IsCharacterOccluded() -> bool
- IsTargetOccluded() -> bool
- GetCollisionAdjustedDistance() -> float

子系统访问：
- GetModuleRegistry() -> UCameraModuleRegistry*
- GetModifierManager() -> UCameraModifierManager*

## 核心架构

### 8阶段Pipeline

```
┌─────────────────────────────────────────────────────────────┐
│                    INPUT LAYER (输入层)                      │
├─────────────────────────────────────────────────────────────┤
│  Stage 1: InputGather      - 收集玩家输入和游戏状态           │
├─────────────────────────────────────────────────────────────┤
│                   DECISION LAYER (决策层)                    │
├─────────────────────────────────────────────────────────────┤
│  Stage 2: StateMachine     - 评估状态转换条件                 │
├─────────────────────────────────────────────────────────────┤
│                   COMPUTE LAYER (计算层)                     │
├─────────────────────────────────────────────────────────────┤
│  Stage 3: ModuleCompute    - 执行39个活跃模块计算             │
│  Stage 4: ModifierApply    - 应用26个修改器效果 (可跳过)      │
│  Stage 5: BlendSolve       - 混合多个输出并解决冲突           │
├─────────────────────────────────────────────────────────────┤
│                    SAFETY LAYER (安全层)                     │
├─────────────────────────────────────────────────────────────┤
│  Stage 6: CollisionResolve - 执行20个碰撞策略 (可跳过)        │
├─────────────────────────────────────────────────────────────┤
│                   OUTPUT LAYER (输出层)                      │
├─────────────────────────────────────────────────────────────┤
│  Stage 7: PostProcess      - 后处理调整 (可跳过)              │
│  Stage 8: RenderApply      - 应用到SpringArm和Camera         │
└─────────────────────────────────────────────────────────────┘
```

## 使用方式

### 基础设置

1. 将`USoulsCameraManager`组件添加到玩家角色
2. 配置`SpringArmComponent`和`CameraComponent`引用
3. 设置`StatesDataTable`（725个状态配置）
4. 系统在BeginPlay时自动初始化所有子系统

### 蓝图调用示例

```cpp
// 获取相机管理器
USoulsCameraManager* Manager = USoulsCameraBlueprintLibrary::GetSoulsCameraManager(this, 0);

// 状态切换（使用FName）
USoulsCameraBlueprintLibrary::RequestCameraStateChange(this, FName("Combat"), false, 0);

// 锁定目标
USoulsCameraBlueprintLibrary::SetCameraLockOnTarget(this, EnemyActor, 0);

// 触发震动（使用ECameraModifierID）
USoulsCameraBlueprintLibrary::TriggerCameraShakeByID(this, ECameraModifierID:: Modifier_S02_Shake_Medium, 1.0f, 0);

// 触发慢动作
USoulsCameraBlueprintLibrary::TriggerCameraSlowMotion(this, 1.0f, 0.3f, 0);

// 触发受击反应
USoulsCameraBlueprintLibrary::TriggerCameraHitReaction(this, true, 1.0f, HitDirection, 0);

// 查询碰撞状态
bool bColliding = USoulsCameraBlueprintLibrary::IsCameraCollisionActive(this, 0);
```

### C++调用示例

```cpp
// 获取Manager
USoulsCameraManager* CameraManager = Character->FindComponentByClass<USoulsCameraManager>();

// 状态切换（使用FName）
CameraManager->RequestStateChange(FName("Combat"), false);

// 获取当前状态
FName CurrentState = CameraManager->GetCurrentStateName();
ECameraStateCategory Category = CameraManager->GetCurrentCategory();

// 锁定目标
CameraManager->SetLockOnTarget(EnemyActor);
CameraManager->ClearLockOnTarget();

// 触发修改器效果
CameraManager->TriggerCameraShake(ECameraModifierID::Modifier_S02_Shake_Medium, 1.5f);
CameraManager->TriggerHitReaction(true, 1.0f, HitDirection);
CameraManager->TriggerSlowMotion(1.0f, 0.3f);
CameraManager->TriggerHitStop(1.0f);
CameraManager->TriggerLowHealthEffect(0.2f);

// 碰撞查询
bool bColliding = CameraManager->IsCollisionActive();
bool bRecovering = CameraManager->IsInCollisionRecovery();
float SafeDistance = CameraManager->GetCollisionAdjustedDistance();

// 获取当前相机数据
FSoulsCameraOutput Output = CameraManager->GetCurrentOutput();
FVector CameraLocation = Output.GetCameraLocation();
FRotator CameraRotation = Output. Rotation;
float Distance = Output.Distance;
float FOV = Output.FOV;

// 调试
FString DebugInfo = CameraManager->GetDebugString();
```

## 扩展指南

### 添加新模块

1. 继承`UCameraModuleBase`
2. 实现`GetModuleType()`返回新的`ECameraModuleType`枚举值
3. 实现`GetModuleCategory()`返回模块分类
4. 实现`Compute()`计算相机输出
5. 在`CameraModuleRegistry:: CreateAllModules()`中注册
6. 在状态配置中激活新模块

### 添加新修改器

1. 继承`UCameraModifierBase`
2. 实现`GetModifierID()`返回新的`ECameraModifierID`枚举值
3. 实现`ComputeEffect()`计算效果
4. 在`CameraModifierManager::CreateAllModifiers()`中注册
5. 添加便捷触发方法到`SoulsCameraManager`

### 添加新碰撞策略

1. 继承`UCameraCollisionBase`
2. 实现`GetStrategyID()`返回新的`ECollisionStrategyID`枚举值
3. 实现`Execute()`执行策略逻辑
4. 在`CameraCollisionResolver:: CreateAllStrategies()`中注册

│  S01: Shake_Hit_Light (轻击震动)                                            │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  触发条件：玩家受到轻攻击伤害                                                │
│                                                                              │
│  震动计算（Perlin噪声）：                                                    │
│  ─────────────────────────────────────────────────────────────────────────  │
│  NoiseTime = ElapsedTime × ShakeFrequency                                   │
│                                                                              │
│  PositionShake. X = PerlinNoise1D(NoiseTime) × Intensity × PosScale         │
│  PositionShake. Y = PerlinNoise1D(NoiseTime + 100) × Intensity × PosScale   │
│  PositionShake.Z = PerlinNoise1D(NoiseTime + 200) × Intensity × PosScale   │
│                                                                              │
│  RotationShake. Pitch = PerlinNoise1D(NoiseTime + 300) × Intensity × RotScale│
│  RotationShake. Yaw = PerlinNoise1D(NoiseTime + 400) × Intensity × RotScale │
│                                                                              │
│  参数：                                                                      │
│  · ShakeFrequency:  25. 0                                                     │
│  · PositionShakeScale: 3.0 cm                                               │
│  · RotationShakeScale: 0.5 degree                                           │
│  · Intensity: 1.0 (由触发事件传入)                                          │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  S02: Shake_Hit_Heavy (重击震动)                                            │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  触发条件：玩家受到重攻击伤害                                                │
│                                                                              │
│  与S01的区别：                                                               │
│  · 震动幅度更大                                                              │
│  · 持续时间更长                                                              │
│  · 附带FOV冲击（短暂缩小FOV）                                               │
│                                                                              │
│  FOV冲击公式：                                                               │
│  ─────────────────────────────────────────────────────────────────────────  │
│  FOVPunch = -Intensity × FOVPunchScale × CurrentWeight                      │
│  （负值 = FOV缩小，产生"被击中收缩"的感觉）                                 │
│                                                                              │
│  参数：                                                                      │
│  · ShakeFrequency: 30.0                                                     │
│  · PositionShakeScale: 8.0 cm                                               │
│  · RotationShakeScale: 1.5 degree                                           │
│  · FOVPunchScale:  5.0 degree                                                │
│                                                                              │
│  参考：血源诅咒的重击反馈                                                    │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘

│  触发条件：                                                                  │
│  · Boss脚步落地                                                              │
│  · 地震事件                                                                  │
│  · 大型爆炸                                                                  │
│  · 建筑倒塌                                                                  │
│                                                                              │
│  特殊处理：                                                                  │
│  · 震动强度随距离衰减                                                        │
│  · 可配置震动源位置                                                          │
│  · 支持连续触发（Boss持续行走）                                              │
│                                                                              │
│  距离衰减公式：                                                              │
│  ─────────────────────────────────────────────────────────────────────────  │
│  Distance = 玩家到震动源的距离                                               │
│  Falloff = 1 - Clamp(Distance / MaxDistance, 0, 1)                          │
│  ActualIntensity = BaseIntensity × Falloff                                  │
│                                                                              │
│  参数：                                                                      │
│  · MaxDistance: 2000 cm                                                     │
│  · ShakeFrequency:  15.0（低频震动）                                         │
│  · PositionShakeScale: 5.0 cm                                               │

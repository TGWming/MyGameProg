## 1. 需求背景

### 1.1 视觉目标
还原《剑风传奇》黄金时代篇���影版第三部《降临》中的**日蚀场景**天空效果：

- 🌑 太阳被黑色圆盘吞噬（可选，本期主要做天空覆盖）
- 🩸 天空整体逐渐变成**血红色**
- 🥩 天空逐渐布满**血肉/血管纹理**
- 👁️ 天空逐渐浮现出**眼睛**（多个、慢慢睁开）
- 🌧️ 同时下**血雨**

### 1.2 关键设计点
- **不是一下变化**，而是**逐渐演变**（类似剑风的渐进式恐怖感）
- 演出时长应有节奏感（推荐 20-40 秒）
- 天空覆盖层 + 血雨 + 颜色变化要协调

### 1.3 已有工具
- ✅ **SkyCreator 蓝图工具**：已能控制天空颜色、血雨开关
- ❌ 血肉/眼睛覆盖效果：本期需要新做

### 1.4 开发者已有理解
- 天空颜色 + 血雨 → SkyCreator 处理（已掌握）
- **核心难点 = 血肉/眼睛逐渐布满天空的实现方式**（本文档主攻方向）

## 4. 方案 A：自定义天空穹顶 + 遮罩材质（核心推荐）

### 4.1 原理

在 SkyCreator 之上**叠加一个额外的半透明球体（Sky Dome）**，球体的材质用**遮罩贴图 + 滑动 Alpha 值**控制血肉**逐渐显现**。

```
摄像机视角从内向外看：
   
   [玩家]
      ↓ 看天空
   ┌────────────────────────────┐
   │ Layer 2: 血肉/眼睛球体（半透明，遮罩控制） ← 新加
   ├────────────────────────────┤
   │ Layer 1: SkyCreator 天空（已有）         │
   └────────────────────────────┘
```

### 4.2 资源准备

1. **血肉贴图（Flesh Texture）**  
   一张包覆球体的血肉肌理图（带血管、纹理）
2. **眼睛贴图（Eye Texture）**  
   多个眼睛分布的图层（可单张 / 可粒子代替 / 可 Decal）
3. **遮罩贴图（Mask Texture）⭐ 关键**  
   黑白噪点图，控制"哪里先长出血肉"

### 4.3 材质蓝图核心算法（伪代码）

```glsl
// === 输入参数 ===
float CoverageAmount = 0.0; // 0 = 完全透明，1 = 完全覆盖
Texture2D FleshAlbedo;      // 血肉颜色
Texture2D EyeAlbedo;        // 眼睛
Texture2D Noise_Mask;       // ⭐ 关键：噪点遮罩（控制生长边界）

// === 核心算法：基于遮罩的渐进显现 ===
float MaskValue = sample(Noise_Mask, UV).r;

// 关键公式：用 step 比较 CoverageAmount 与 MaskValue
float Visibility = saturate((CoverageAmount - MaskValue) * 10.0);

// 边界过渡（让边缘有"溶解"感而不是硬切）
Visibility = smoothstep(MaskValue - 0.05, MaskValue + 0.05, CoverageAmount);

// === 最终颜色混合 ===
float3 FleshColor = sample(FleshAlbedo, UV).rgb;
float3 EyeColor = sample(EyeAlbedo, UV).rgb;

Emissive = FleshColor + EyeColor * EyeIntensity;
Opacity = Visibility;
```

### 4.4 渐进显现的核心原理

```
CoverageAmount = 0.0  →  完全无血肉（透明）
CoverageAmount = 0.3  →  噪点最深的地方先显现血肉
CoverageAmount = 0.6  →  血肉扩散到中间区域
CoverageAmount = 1.0  →  完全覆盖整个天空
```

> 这是**溶解 Shader 的反向应用** —— 不是消失，而是**逐渐显现**。  
> 视觉效果就是"血肉慢慢从天空缝隙中渗透出来"🩸

### 4.5 蓝图控制结构

```
BP_BloodSkyDome (Actor)
   ├─ Static Mesh: SM_SkyDome（反向法线的球体）
   ├─ Material Instance: MI_BloodSky（暴露 CoverageAmount 参数）
   │
   └─ 蓝图函数:
       PlayEclipseTransition(float Duration = 30.0)
          ↓
       Timeline (0 → 1 over Duration)
          ↓
       SetScalarParameterValue("CoverageAmount", CurrentValue)
          ↓
       天空逐渐被血肉覆盖 ✅
```

## 5. 方案 B：多层球体堆叠（细腻层次）

### 5.1 适用场景
追求**电影级戏剧节奏感**，让效果有"递进式戏剧性"而非平滑过渡。

### 5.2 结构

```
最外层球体: 血雾/红色滤镜    (CoverageAmount 控制)
   ↓
中间球体:   血肉肌理         (LayerMask 1 控制)
   ↓
内层球体:   眼睛漂浮         (LayerMask 2 控制 + 旋转动画)
   ↓
SkyCreator: 基础天空
```

### 5.3 时间偏移示例
- **0-10s**：天空变红（最外层）
- **5-25s**：血肉开始生长（中间层）
- **15-30s**：眼睛逐渐睁开（内层）

### 5.4 优劣
- ✅ 节奏感强，更具戏剧性
- ❌ 性能成本略高（多层透明叠加）
- ❌ 需要协调多个材质参数

## 6. 方案 C：眼睛用粒子系统（动态感最强）

### 6.1 原理
血肉用方案 A 的球体材质，**眼睛单独用 Niagara/Cascade 粒子系统**生成。

### 6.2 优势
- ✅ 每个眼睛**独立生成、独立闭合/睁开动画**
- ✅ 可做**眼睛追踪玩家**的恐怖效果（剑风式被注视感）
- ✅ 数量、位置可随机化
- ✅ 易于调整密度和分布

### 6.3 实现思路

```
NiagaraSystem: NS_SkyEyes
   ├─ 发射器: 在球面上随机生成位置（半径 = 天空盒半径）
   ├─ 粒子: 每个粒子是一个 Sprite/Mesh（眼睛模型/贴图）
   ├─ 生命周期: 永久（除非清除）
   ├─ 出生动画: 0 → 完整眼球（缩放 + Alpha 渐变）
   ├─ 行为: 缓慢旋转 / 偶尔眨眼（材质参数动画）
   └─ 数量随时间增加（用 Spawn Rate Curve）
```

### 6.4 SpawnRate 曲线示例

```
0s:  0 个/秒
5s:  1 个/秒
15s: 5 个/秒
30s: 20 个/秒（上限 200 个）
```

效果：眼睛**不是一次性出现**，而是**慢慢一颗一颗睁开** 👁️👁️👁️

## 7. 方案 D：环境光照同步血色化

### 7.1 必要性
天空变了，**地面也要被血红光照亮**才像剑风。否则会有"天空假，地面正常"的违和感。

### 7.2 实现要点

1. **Directional Light**
   - 颜色 Lerp 到深红色
   - 强度可适当降低
2. **Sky Light**
   - 重新捕获（捕获新的红色天空 → 间接光自动变红）
   - 或用 Specified Cubemap 直接换贴图
3. **PostProcessVolume**
   - Color Grading → Shadows 加红 / Midtones 加红
   - Vignette → 0.7（强化压迫感）
   - Bloom → 1.5（光晕扩散）
   - Saturation → 略降（让画面更阴暗）

## 11. 进阶细节（让效果更"剑风"）

### 11.1 眼睛追踪玩家
让某些眼球的瞳孔朝向玩家，**Shader 实现**：

```glsl
float3 ToPlayer = normalize(PlayerPos - WorldPos);
UVOffset = ToPlayer.xy * EyeMaxRotation;
PupilUV = BaseUV + UVOffset;
```

### 11.2 血肉脉动
材质里加 `sin(Time)` 让血肉**呼吸般微微起伏**：

```glsl
float Pulse = sin(Time * 2.0) * 0.05;
WorldPositionOffset = Normal * Pulse;
```

### 11.3 撕裂动画
某些区域可以做**裂开 → 露出眼睛**的演出：
- 噪点遮罩用动态贴图（Flipbook）
- 或者用 World Position Offset 模拟撕开

### 11.4 音效配合
- 心跳声（低频，与脉动同步）
- 肉块蠕动声
- 远处的低吼
- 演出高潮时配合 Stinger（接入现有音频系统）

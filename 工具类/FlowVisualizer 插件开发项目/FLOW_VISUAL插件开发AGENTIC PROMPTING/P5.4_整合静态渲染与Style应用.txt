【任务目标】
统一整合 SFlowVisualizerPanel 中所有 OnPaint 绘制逻辑，
将之前任务中的硬编码颜色和字体全部替换为 FlowVisStyle:: 常量，
并确认绘制顺序正确、窗口缩放自适应。

【具体要求】

1. 需要修改的文件：

   Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Private/SFlowVisualizerPanel.cpp

2. 添加 include：
   - #include "FlowVisualizerStyle.h"

3. 替换所有硬编码颜色为 FlowVisStyle 常量：

   背景绘制（P3.2 / 整体）：
   - FLinearColor(0.05f, 0.05f, 0.08f, 1.0f)
     → FlowVisStyle::Background

   顶栏背景（P4.2 Construct 中）：
   - FLinearColor(0.10f, 0.10f, 0.15f, 1.0f)
     → FlowVisStyle::TitleBarBg

   节点文字颜色（P5.1）：
   - FLinearColor(0.40f, 0.40f, 0.40f, 1.0f)
     → FlowVisStyle::NodeInactive

   连线颜色（P5.2）：
   - FLinearColor(0.53f, 0.53f, 0.53f, 1.0f)
     → FlowVisStyle::EdgeDefault

   底栏背景（P4.3）：
   - FLinearColor(0.10f, 0.10f, 0.15f, 1.0f)
     → FlowVisStyle::TitleBarBg

   底栏文字（P4.3）：
   - FLinearColor(0.60f, 0.60f, 0.60f, 1.0f)
     → FlowVisStyle::StatusText

4. 替换所有硬编码字体为 FlowVisStyle 函数：

   节点文字字体（P5.1）：
   - FCoreStyle::GetDefaultFontStyle("Bold", 14)
     → FlowVisStyle::NodeTitleFont()

   底栏文字字体（P4.3）：
   - FCoreStyle::GetDefaultFontStyle("Mono", 10)
     → FlowVisStyle::StatusBarFont()

   如果顶栏有文字使用字体：
   - → FlowVisStyle::PipelineTitleFont()

5. 确认并调整 OnPaint 绘制顺序（从底到顶）：

   LayerId 基准值（传入参数）：
   Layer 0: 背景矩形（FlowVisStyle::Background）
   Layer 1: 连线 + 箭头（FlowVisStyle::EdgeDefault）
   Layer 2: 节点文字（FlowVisStyle::NodeInactive）
   Layer 3: 底栏背景（FlowVisStyle::TitleBarBg）
   Layer 4: 底栏文字（FlowVisStyle::StatusText）

   确保连线在节点下方，底栏在所有内容上方。

6. 确认窗口缩放自适应：
   - 调整窗口大小时 NodeRelativeToPixel() 基于实时 Geometry 计算
   - 连线端点跟随节点位置更新
   - 底栏始终贴��，顶栏始终贴顶
   - 如果发现缩放问题，修复坐标计算

7. 关键技术注意事项：
   - 替换过程是纯文本搜索替换，不改变任何绘制逻辑
   - Construct() 中 SBorder 的 BorderBackgroundColor 也需要替换
   - 确保 #include "FlowVisualizerStyle.h" 路径正确
     （文件在 Private 目录，同模块内可直接引用）

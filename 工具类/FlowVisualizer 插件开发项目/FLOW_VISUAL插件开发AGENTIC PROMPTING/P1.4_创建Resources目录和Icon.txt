【任务目标】
在插件 Resources 目录下创建一个 128x128 的 PNG 占位图标，
供 UE4 插件管理器显示使用。

【具体要求】

1. 目标文件：Plugins/FlowVisualizer/Resources/Icon128.png

2. 图标规格：
   - 尺寸：128 x 128 像素
   - 格式：PNG
   - 内容：可以是以下任一方案：
     a. 纯色深蓝/黑色背景（#0D0D14 或近似）+ 白色 "FV" 字样居中
     b. 使用 UE4 默认插件图标格式的简单占位图
     c. 任何合法的 128x128 PNG 即可（这是占位图标）

3. 关键说明：
   - UE4 插件系统会在 Plugin Browser 中查找 Resources/Icon128.png
   - 如果 IDE Agent 无法生成二进制 PNG 文件，可以：
     a. 使用代码生成一个最小合法 PNG（纯色 128x128）
     b. 或者创建一个说明文件 Resources/README_ICON.txt 说明需要
        手动放置一个 128x128 PNG 图标，并命名为 Icon128.png
   - 图标不影响插件功能，仅影响 Plugin Browser 显示

插件采用双模块结构：FlowVisualizerRuntime + FlowVisualizerEditor。

【任务目标】
修改 SFlowVisualizerPanel 的 Construct()，在面板顶部添加一个工具栏区域，
包含 Pipeline 下拉选择菜单、Pause 按钮和 Clear 按钮。
下拉菜单的数据源来自 FFlowPipelineRegistry，选择后更新 CurrentPipelineID。

【具体要求】

1. 需要修改的文件：

   文件 A：Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Public/SFlowVisualizerPanel.h
   文件 B：Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Private/SFlowVisualizerPanel.cpp

2. SFlowVisualizerPanel.h 中应新增以下成员声明：

   数据成员：
   - FName CurrentPipelineID
     （当前选中的管线 ID）
   - TArray<TSharedPtr<FName>> PipelineIDList
     （下拉菜单的数据源，TSharedPtr 包装供 SComboBox 使用）
   - TSharedPtr<FName> SelectedPipelineItem
     （当前选中项）
   - bool bIsPaused = false
     （暂停标志，后续 P6.1 / P9.1 使用）
   - FDelegateHandle RegistryChangedHandle
     （监听 Registry 变化的 Delegate 句柄）

   函数成员：
   - void RefreshPipelineList()
     （从 Registry 获取最新列表，刷新下拉菜单数据源）
   - void OnPipelineSelected(TSharedPtr<FName> NewSelection, ESelectInfo::Type)
     （下拉菜单选择回调）
   - TSharedRef<SWidget> GeneratePipelineComboItem(TSharedPtr<FName> Item)
     （下拉菜单每一项的 Widget 生成）
   - FText GetSelectedPipelineText() const
     （获取当前选中项的显示文字）

3. Construct() 应修改为：

   整体布局改为 SVerticalBox，从上到下：
   
   槽位 1 — 顶栏工具条（AutoHeight）：
   - 背景色：FLinearColor(0.10f, 0.10f, 0.15f, 1.0f)  即 #1A1A25
   - 内部为 SHorizontalBox，包含：
     a. STextBlock "Pipeline:" 白色文字
     b. SComboBox<TSharedPtr<FName>>
        - OptionsSource: &PipelineIDList
        - OnSelectionChanged: 绑定 OnPipelineSelected
        - OnGenerateWidget: 绑定 GeneratePipelineComboItem
        - Content: 显示 GetSelectedPipelineText()
     c. 间距 Spacer
     d. SButton "Pause"
        - 点击切换 bIsPaused（本任务仅切换标志 + 打印日志占位）
     e. SButton "Clear"
        - 点击调用 FFlowTracer::Get().ClearBuffer()
        （需要 #include "FlowTracer.h"）
   - 顶栏高度建议约 30~40 像素

   槽位 2 — 主画布区域（FillHeight(1.0f)）：
   - 保留原有的 SOverlay（OnPaint 绘制区域）

4. Construct() 末尾应注册 Registry 监听：
   - RegistryChangedHandle = FFlowPipelineRegistry::Get()
       .OnRegistryChanged.AddRaw(this, &...::RefreshPipelineList)
   - 立即调用一次 RefreshPipelineList() 初始化列表

5. 析构/销毁时应取消监听：
   - FFlowPipelineRegistry::Get().OnRegistryChanged.Remove(RegistryChangedHandle)

6. RefreshPipelineList() 实现：
   - 从 FFlowPipelineRegistry::Get().GetAllPipelineIDs() 获取列表
   - 清空 PipelineIDList，重新填充为 TSharedPtr<FName> 包装
   - 如果 CurrentPipelineID 不在新列表中且列表非空，
     自动选中第一项
   - 如果列表为空，CurrentPipelineID 置为 NAME_None

7. 关键技术注意事项：
   - SComboBox 需要 #include "Widgets/Input/SComboBox.h"
   - SButton 需要 #include "Widgets/Input/SButton.h"
   - STextBlock 需要 #include "Widgets/Text/STextBlock.h"
   - FFlowPipelineRegistry 需要 #include "FlowPipelineRegistry.h"
   - FFlowTracer 需要 #include "FlowTracer.h"（Clear 按钮使用）
   - 顶栏背景可用 SBorder 包裹并设置 BorderBackgroundColor

【此任务不做的事情】
- 不实现底栏状态条（那是 P4.3）
- 不实现节点/连线绘制（那是 P5.x）
- 不实现 Pause/Resume 的完整逻辑（那是 P9.1，本任务仅切换 bool 标志）
- 不连接 FFlowTracer 的事件 Delegate（那是 P6.1）

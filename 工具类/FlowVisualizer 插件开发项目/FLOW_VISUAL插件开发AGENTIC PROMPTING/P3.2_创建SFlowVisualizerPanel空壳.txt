【任务目标】
在 FlowVisualizerEditor 模块中创建 SFlowVisualizerPanel Widget，
继承 SCompoundWidget，提供终端风格的黑色面板基底。
该阶段仅实现空壳：黑色背景 + 一行白色占位文字。
后续任务会在此基础上逐步添加绘制逻辑。

【具体要求】

1. 需要创建 2 个文件：

   文件 A：Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Public/SFlowVisualizerPanel.h
   文件 B：Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Private/SFlowVisualizerPanel.cpp

2. SFlowVisualizerPanel 类应满足：
   - 继承 SCompoundWidget
   - 使用 SLATE_BEGIN_ARGS / SLATE_END_ARGS 标准宏

3. Construct() 应实现：
   - ChildSlot 包含一个 SBox（或 SBorder）
   - 背景色设置为 FLinearColor(0.05f, 0.05f, 0.08f, 1.0f)
     （纯黑微蓝，对应设计规格中的 #0D0D14）
   - SBox 内部放一个 SOverlay（供后续绘制层叠加使用）

4. Tick(const FGeometry&, double, float) 应声明并提供空实现：
   - override 父类 Tick
   - 函数体暂时为空，后续 P6.4 会填充 Invalidate 逻辑

5. OnPaint() 应实现基础绘制：
   - override 父类 OnPaint，返回 int32 LayerId
   - 绘制黑色背景矩形：
     使用 FSlateDrawElement::MakeBox 填充整个 AllottedGeometry
     颜色：FLinearColor(0.05f, 0.05f, 0.08f, 1.0f)
   - 绘制一行白色占位文字 "Flow Visualizer - Ready"：
     使用 FSlateDrawElement::MakeText
     字体：FCoreStyle::GetDefaultFontStyle("Mono", 14)
     颜色：FLinearColor::White
     位置：面板中央偏上区域
   - 返回 LayerId + 1

6. 关键技术注意事项：
   - SCompoundWidget 的 OnPaint 签名（UE4.27）：
     virtual int32 OnPaint(const FPaintArgs& Args,
       const FGeometry& AllottedGeometry,
       const FSlateRect& MyCullingRect,
       FSlateDrawElement& OutDrawElements,
       int32 LayerId,
       const FWidgetStyle& InWidgetStyle,
       bool bParentEnabled) const override;
   - FSlateDrawElement 静态函数需要 #include "Rendering/DrawElements.h"
   - FCoreStyle 需要 #include "Styling/CoreStyle.h"
   - WhiteBrush 可通过 FCoreStyle::Get().GetBrush("WhiteBrush") 获取

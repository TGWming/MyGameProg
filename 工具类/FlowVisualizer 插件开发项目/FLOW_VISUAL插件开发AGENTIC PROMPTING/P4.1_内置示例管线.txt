【任务目标】
在 FlowVisualizerEditorModule 的 StartupModule() 末尾注册一个内置的
"Demo" 管线，供开发和测试使用。
该管线包含 3 个节点和 2 条连线，形成简单的线性流：
Input -> Process -> Output

【具体要求】

1. 需要修改的文件：

   Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Private/FlowVisualizerEditorModule.cpp

2. 在 StartupModule() 末尾（菜单注册之后）添加 Demo 管线注册逻辑：

   构造一个 FFlowPipeline 实例，属性如下：
   - PipelineID: "Demo"
   - DisplayName: "Demo Pipeline"

   包含 3 个 FFlowNode：
   - NodeID: "Input",   DisplayName: "Input",   RelativePosition: (0.10, 0.40)
   - NodeID: "Process", DisplayName: "Process", RelativePosition: (0.40, 0.40)
   - NodeID: "Output",  DisplayName: "Output",  RelativePosition: (0.70, 0.40)

   包含 2 条 FFlowEdge：
   - SourceNodeID: "Input",   TargetNodeID: "Process"
   - SourceNodeID: "Process", TargetNodeID: "Output"

   调用 FFlowPipelineRegistry::Get().RegisterPipeline(Demo) 完成注册。

3. 需要添加的 include：
   - #include "FlowPipelineTypes.h"
   - #include "FlowPipelineRegistry.h"

4. 关键说明：
   - RelativePosition 范围 (0~1)，表示节点在画布上的相对位置
   - 三个节点 Y 坐标相同 (0.40)，形成水平排列
   - X 坐标分别为 0.10, 0.40, 0.70，均匀分布
   - Demo 管线是插件自带的测试管线，���续 Phase 8 会注册
     Soul 项目的 4 条正式管线

【任务目标】
修改 SFlowVisualizerPanel 的 OnPaint()，根据当前选中的管线
（CurrentPipelineID），从 Registry 获取管线数据，
遍历所有节点并将节点名以终端风格 "[NodeName]" 绘制在画布上。
节点位置基于 RelativePosition 转换为像素坐标。

【具体要求】

1. 需要修改的文件：

   文件 A：Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Public/SFlowVisualizerPanel.h
   文件 B：Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Private/SFlowVisualizerPanel.cpp

2. SFlowVisualizerPanel.h 中应新增：

   常量/配置��
   - static const float CanvasMargin = 40.0f
     （画布四周边距，防止��点贴边）
   - static const float TopBarHeight = 36.0f
     （顶栏高度，与 P4.2 中顶栏实际高度匹配）

   辅助函数：
   - FVector2D NodeRelativeToPixel(const FVector2D& RelativePos,
       const FGeometry& Geometry) const
     （将 RelativePosition (0~1) 转换为画布像素坐标）

3. NodeRelativeToPixel() 实现逻辑：
   - CanvasWidth = Geometry.GetLocalSize().X
   - CanvasHeight = Geometry.GetLocalSize().Y
   - PixelX = RelativePos.X * (CanvasWidth - CanvasMargin * 2) + CanvasMargin
   - PixelY = RelativePos.Y * (CanvasHeight - TopBarHeight - StatusBarHeight
     - CanvasMargin) + TopBarHeight + CanvasMargin * 0.5f
   - 返回 FVector2D(PixelX, PixelY)

4. OnPaint() 中应在背景绘制之后、底栏绘制之前添加节点绘制逻辑：

   步骤 1 — 获取当前管线数据：
   - 通过 FFlowPipelineRegistry::Get().FindPipeline(CurrentPipelineID)
   - 如果返回 nullptr，跳过节点绘制

   步骤 2 — 遍历管线的所有 Nodes：
   - 对每个 FFlowNode：
     a. 调用 NodeRelativeToPixel() 获取像素坐标
     b. 组装显示文字：FString::Printf(TEXT("[%s]"), *Node.DisplayName.ToString())
        方括号 [] 模拟终端风格
     c. 使用 FSlateDrawElement::MakeText 绘制
     d. 字体：FCoreStyle::GetDefaultFontStyle("Bold", 14)
        （Roboto Bold 14pt，后续 P5.3 会替换为 FlowVisStyle 常量）
     e. 颜色：FLinearColor(0.40f, 0.40f, 0.40f, 1.0f) 即 #666666
        （默认非活跃暗灰色，后续 P6.2 会根据高亮状态动态计算）
     f. 位置：像素坐标处

   步骤 3 — 节点下方预留数据文字行空间：
   - 本任务不绘制数据文字（那是 P6.3）
   - 但节点位置计算应考虑后续数据行的空间
     （留注释说明即可，不实现）

5. 关键技术注意事项：
   - OnPaint 中的 FSlateDrawElement::MakeText 签名（UE4.27）：
     MakeText(OutDrawElements, LayerId, PaintGeometry, Text, Font, 
              DrawEffects, Color)
   - PaintGeometry 可通过 AllottedGeometry.ToPaintGeometry(Position, Size)
     或 AllottedGeometry.ToPaintGeometry() 后偏移
   - 文字位置需要创建偏移后的 FPaintGeometry：
     AllottedGeometry.MakeChild(FVector2D(PixelX, PixelY), FVector2D(200, 30))
     然后 .ToPaintGeometry()
   - 需要 #include "FlowPipelineRegistry.h"

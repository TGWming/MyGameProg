【任务目标】
修改 SFlowVisualizerPanel 的 OnPaint()，在面板底部绘制一个
固定高度的状态栏，显示实时帧号、FPS 和当前管线名称。
后续 Phase 会在此基础上扩展更多统计信息。

【具体要求】

1. 需要修改的文件���

   文件 A：Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Public/SFlowVisualizerPanel.h
   文件 B：Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Private/SFlowVisualizerPanel.cpp

2. SFlowVisualizerPanel.h 中应新增：
   - static const float StatusBarHeight = 30.0f
     （底栏高度常量，或放在 cpp 中作为局部常量）
   - float CachedFPS = 0.0f
     （缓存的 FPS 值，避免每帧计算抖动过大，可做简单平滑）

3. OnPaint() 中应在绘制末尾添加底栏绘制逻辑：

   步骤 1 — 计算底栏区域：
   - StatusBarY = AllottedGeometry.GetLocalSize().Y - StatusBarHeight
   - StatusBarRect: 从 (0, StatusBarY) 到 (Width, Height)

   步骤 2 — 绘制底栏背景：
   - FSlateDrawElement::MakeBox
   - 颜色：FLinearColor(0.10f, 0.10f, 0.15f, 1.0f)  即 #1A1A25
   - 覆盖整个底栏区域

   步骤 3 — 组装状态文字：
   - 格式："Frame: {N} | FPS: {N} | Pipeline: {name}"
   - Frame 值：GFrameNumber
   - FPS 值：从 Tick 的 DeltaTime 计算，取整显示
     简单平滑：CachedFPS = FMath::Lerp(CachedFPS, 1.0f/DeltaTime, 0.1f)
   - Pipeline 名称：CurrentPipelineID.ToString()
     如果 CurrentPipelineID 为 NAME_None 则显示 "None"
   - 后续 P9.2 会扩展为完整统计：
     "Frame: {N} | FPS: {N} | Active: {N}/{Total} | Last: {N}ms ago | Buffer: {N}/{Cap}"
     目前只实现基础三项即可

   步骤 4 — 绘制状态文字：
   - FSlateDrawElement::MakeText
   - 字体：FCoreStyle::GetDefaultFontStyle("Mono", 10)
   - 颜色：FLinearColor(0.60f, 0.60f, 0.60f, 1.0f)  即 #999999
   - 位置：底栏内左侧，垂直居中，水平留 10px 左边距

4. Tick() 中应新增 FPS 缓存更新：
   - 如果 InDeltaTime > 0：
     CachedFPS = FMath::Lerp(CachedFPS, 1.0f / InDeltaTime, 0.1f)
   - 注意：Tick 已在 P3.2 中声明为空实现，本任务添加 FPS 计算

5. 关键技术注意事项：
   - OnPaint 中 AllottedGeometry ��坐标系原点在 Widget 左上角
   - 底栏绘制应在最高 LayerId 上，确保覆盖主画布内容
   - GFrameNumber 是全局变量，直接引用即可
   - 主画布区域的 OnPaint 绘制（P3.2 中的占位文字）应考虑
     底栏占用的空间，避免文字绘制在底栏区域内
     （可在绘制时将 Canvas 有效区域减去 StatusBarHeight）

【任务目标】
修改 SFlowVisualizerPanel 的 OnPaint()，在节点文字绘制之前
绘制管线中所有连线（Edge），包括线段和箭头头部。
连线绘制在节点下方（较低 LayerId）���确保节点文字在连线上方。

【具体要求】

1. 需要修改的文件：

   Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Private/SFlowVisualizerPanel.cpp

2. OnPaint() 中应在节点绘制之前添加连线绘制逻辑：

   绘制顺序确认（从底到顶）：
   a. 背景矩形
   b. 连线 + 箭头（本任务）   ← LayerId
   c. 节点文字（P5.1）        ← LayerId + 1
   d. 底栏（P4.3）            ← LayerId + 2

3. 遍历当前管线的所有 Edges，对每条 Edge：

   步骤 1 — 获取起止节点的像素坐标：
   - 在管线 Nodes 数组中查找 SourceNodeID 和 TargetNodeID 对应的节点
   - 调用 NodeRelativeToPixel() 获取各自像素坐标
   - 如果任一节点未找到，跳过该 Edge

   步骤 2 — 计算连线端点偏移：
   - Source 点：SourceNode 像素坐标右侧偏移
     SourcePoint = SourcePixel + FVector2D(TextWidth * 0.5f + 10.0f, 8.0f)
     （文字右端中心附近，8.0f 为近似文字半高）
   - Target 点：TargetNode 像素坐标左侧偏移
     TargetPoint = TargetPixel + FVector2D(-10.0f, 8.0f)
     （文字左端中心附近）
   - TextWidth 估算：节点名字符数 * 8.0f（粗略估算，
     或使用固定偏移 50.0f 简化处理）

   步骤 3 — 绘制连线：
   - 使用 FSlateDrawElement::MakeLines
   - 构造 TArray<FVector2D> 包含 SourcePoint 和 TargetPoint
   - 颜色：FLinearColor(0.53f, 0.53f, 0.53f, 1.0f) 即 #888888（中灰）
     （后续 P5.4 会替换为 FlowVisStyle 常量）
   - 线宽（Thickness）：2.0f
   - bAntialias：true

   步骤 4 — 绘制箭头��部：
   - 在 TargetPoint 端绘制箭头
   - 方法：用两条短线段模拟 ">" 形箭头
   - 计算连线方向向量：Dir = (TargetPoint - SourcePoint).GetSafeNormal()
   - 箭头两翼：
     ArrowSize = 8.0f
     Perpendicular = FVector2D(-Dir.Y, Dir.X)（垂直于连线方向）
     Wing1 = TargetPoint - Dir * ArrowSize + Perpendicular * ArrowSize * 0.5f
     Wing2 = TargetPoint - Dir * ArrowSize - Perpendicular * ArrowSize * 0.5f
   - 绘制两条线段：
     MakeLines: Wing1 -> TargetPoint
     MakeLines: Wing2 -> TargetPoint
   - 颜色和线宽与连线一致

   步骤 5 — 处理垂直/非水平连线：
   - 上述算法对任意角度的连线均适用（基于方向向量计算）
   - 如果节点上下排列（X 坐标接近），连线为近垂直线，
     箭头自动朝下，无需特殊处理

4. 关键技术注意事项：
   - FSlateDrawElement::MakeLines 签名（UE4.27）：
     MakeLines(OutDrawElements, LayerId, PaintGeometry,
              Points, DrawEffects, Color, bAntialias, Thickness)
   - Points 参数为 TArray<FVector2D>
   - PaintGeometry 使用 AllottedGeometry.ToPaintGeometry()
   - 线段坐标是相对于 Widget 左上角的本地坐标
   - 箭头绘制也使用 MakeLines（分别两条线段）

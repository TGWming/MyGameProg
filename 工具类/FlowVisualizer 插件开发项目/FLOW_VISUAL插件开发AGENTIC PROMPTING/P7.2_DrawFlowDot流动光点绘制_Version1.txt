【任务目标】
修改 SFlowVisualizerPanel OnPaint() 中的连线绘制逻辑，
根据 EdgeLastActiveTime 实现两个动画效果：
1. Edge 高亮变色：激活的连线从亮青色渐灭回灰色
2. 流动光点：一个白色小方块沿激活连线从 Source 端向 Target 端移动

【具体要求】

1. 需要修改的文件：

   Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Private/SFlowVisualizerPanel.cpp

2. 修改 OnPaint() 中连线绘制部分（P5.2 实现的逻辑），
   对每条 Edge 的绘制逻辑改为：

   步骤 1 — 查找 Edge 激活状态：
   - TPair<FName, FName> EdgeKey(Edge.SourceNodeID, Edge.TargetNodeID)
   - const double* EdgeActivePtr = EdgeLastActiveTime.Find(EdgeKey)
   - double CurrentTime = FPlatformTime::Seconds()
     （复用 OnPaint 入口处已计算的 CurrentTime）

   步骤 2 — 计算连线颜色：
   - 如果 EdgeActivePtr 不为 nullptr：
     double Elapsed = CurrentTime - *EdgeActivePtr
     如果 Elapsed < FlowVisStyle::HighlightFadeDuration：
       float EdgeAlpha = 1.0f - (float)(Elapsed / FlowVisStyle::HighlightFadeDuration)
       LineColor = FMath::Lerp(FlowVisStyle::EdgeDefault, FlowVisStyle::EdgeActive, EdgeAlpha)
     否则：
       LineColor = FlowVisStyle::EdgeDefault
   - 如果 EdgeActivePtr 为 nullptr：
     LineColor = FlowVisStyle::EdgeDefault

   步骤 3 — 绘制连线（使用动态 LineColor）：
   - 与 P5.2 相同的 MakeLines 调用，但颜色替换为计算出的 LineColor
   - 箭头颜色也使用同一 LineColor

   步骤 4 — 绘制流动光点（仅在 Edge 活跃时）：
   - 条件：EdgeActivePtr != nullptr 且 Elapsed < HighlightFadeDuration

   - 计算光点位置：
     SourcePos = SourceNode 的像素坐标（与连线端点一致）
     TargetPos = TargetNode 的像素坐标
     EdgeLength = FVector2D::Distance(SourcePos, TargetPos)

     如果 EdgeLength > 0：
       Progress = FMath::Fmod(
         (float)(Elapsed * FlowVisStyle::FlowDotSpeed / EdgeLength), 1.0f)
       DotPos = FMath::Lerp(SourcePos, TargetPos, Progress)
       DotAlpha = EdgeAlpha（与连线高亮同步衰减）

   - 绘制光点：
     使用 FSlateDrawElement::MakeBox
     位置：DotPos - FVector2D(3.0f, 3.0f)
     大小：FVector2D(6.0f, 6.0f)（6x6 像素白色方块）
     颜色：FLinearColor(1.0f, 1.0f, 1.0f, DotAlpha)
     即 FlowVisStyle::FlowDot 带 Alpha 衰减
     Brush：FCoreStyle::Get().GetBrush("WhiteBrush")（或 FEditorStyle）
     LayerId：连线 LayerId + 1（光点在连线上方）

3. 关键技术注意事项：
   - FSlateDrawElement::MakeBox 签名（UE4.27）：
     MakeBox(OutDrawElements, LayerId, PaintGeometry, Brush, DrawEffects, Color)
   - PaintGeometry 需要通过：
     AllottedGeometry.ToPaintGeometry(DotPos - FVector2D(3,3), FVector2D(6,6))
     或使用 MakeChild 构造子 Geometry
   - WhiteBrush 获取：
     const FSlateBrush* WhiteBrush = FCoreStyle::Get().GetBrush("WhiteBrush")
     可在 OnPaint 入口处缓存，避免每次查找
   - FlowDotSpeed = 200.0f（像素/秒），意味着光点每秒移动 200 像素
   - Fmod 使光点在到达 Target 后循环回 Source（持续移动效果）
   - 多条 Edge 可同时有独立的光点，互不影响
   - 确保连线绘制在节点文字下方，光点在连线上方但在节点文字下方

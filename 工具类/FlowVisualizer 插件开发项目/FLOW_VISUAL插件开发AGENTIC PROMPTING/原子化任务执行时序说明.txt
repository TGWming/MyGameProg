【任务目标】
修改 FlowVisualizerEditorModule 的窗口管理逻辑，使用 UE4 的
GConfig（EditorPerProjectUserSettings）保存和恢复窗口位置和大小。
用户关闭窗口时保存，下次打开时恢复到上次的位置和大小。

【具体要求】

1. 需要修改的文件：

   文件 A：Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Public/FlowVisualizerEditorModule.h
   文件 B：Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Private/FlowVisualizerEditorModule.cpp

2. FlowVisualizerEditorModule.h 中应新增：

   窗口状态保存/恢复函数：
   - void SaveWindowLayout()
   - void LoadWindowLayout(FVector2D& OutPosition, FVector2D& OutSize)

   GConfig Section/Key 常量：
   - static const FString ConfigSection = TEXT("FlowVisualizer.WindowLayout")
   - 或使用 static const TCHAR* 形式

3. SaveWindowLayout() 实现：
   - 如果 VisualizerWindow.IsValid()：
     获取窗口当前位置：FVector2D Pos = VisualizerWindow->GetPositionInScreen()
     获取窗口当前大小：FVector2D Size = VisualizerWindow->GetSizeInScreen()
     （注意：UE4.27 中获取窗口位置/大小的 API 可能是
      GetPositionInScreen() / GetClientSizeInScreen()，
      或 GetRectInScreen()，IDE Agent 需确认实际 API）

   - 使用 GConfig 保存：
     GConfig->SetFloat(TEXT("FlowVisualizer.WindowLayout"),
         TEXT("PosX"), Pos.X,
         GEditorPerProjectIni);
     GConfig->SetFloat(..., TEXT("PosY"), Pos.Y, GEditorPerProjectIni);
     GConfig->SetFloat(..., TEXT("SizeX"), Size.X, GEditorPerProjectIni);
     GConfig->SetFloat(..., TEXT("SizeY"), Size.Y, GEditorPerProjectIni);
     GConfig->Flush(false, GEditorPerProjectIni);

4. LoadWindowLayout() 实现：
   - 使用 GConfig 读取：
     float PosX, PosY, SizeX, SizeY;
     bool bHasLayout = true;
     bHasLayout &= GConfig->GetFloat(TEXT("FlowVisualizer.WindowLayout"),
         TEXT("PosX"), PosX, GEditorPerProjectIni);
     bHasLayout &= GConfig->GetFloat(..., TEXT("PosY"), PosY, ...);
     bHasLayout &= GConfig->GetFloat(..., TEXT("SizeX"), SizeX, ...);
     bHasLayout &= GConfig->GetFloat(..., TEXT("SizeY"), SizeY, ...);

   - 如果 bHasLayout == true：
     OutPosition = FVector2D(PosX, PosY)
     OutSize = FVector2D(SizeX, SizeY)
   - 如果 bHasLayout == false（首次启动，无保存记录）：
     OutPosition = FVector2D(-1, -1)（标记为无效，使用默认居中）
     OutSize = FVector2D(1200, 700)（默认大小）

   - 安全校验：
     OutSize.X = FMath::Clamp(OutSize.X, 400.0f, 3840.0f)
     OutSize.Y = FMath::Clamp(OutSize.Y, 300.0f, 2160.0f)
     （防止异常值）

5. OpenVisualizerWindow() 修改：
   - 在创建 SWindow 之前调用 LoadWindowLayout() 获取位置和大小
   - SNew(SWindow) 的属性修改为：
     .ClientSize(LoadedSize)
     如果 LoadedPosition 有效（不是 -1,-1）：
       .ScreenPosition(LoadedPosition)
       .AutoCenter(EAutoCenter::None)
     否则：
       .AutoCenter(EAutoCenter::PreferredWorkArea)

6. OnWindowClosed() 修改：
   - 在 VisualizerWindow.Reset() 之前调用 SaveWindowLayout()
   - 即先保存再释放窗口引用

7. ShutdownModule() 修改：
   - 在 RequestDestroyWindow() 之前也调用 SaveWindowLayout()

8. 关键技术注意事项：
   - GEditorPerProjectIni 是 UE4 编辑器的项目级用户配置文件
     保存在 Saved/Config/Windows/EditorPerProjectUserSettings.ini
   - GConfig 需要 #include "Misc/ConfigCacheIni.h"
   - GetPositionInScreen() 返回的是屏幕坐标，包含多显示器偏移
   - 如果用户将窗口拖到已断开的显示器位置，下次打开会超出屏幕
     可选：增加屏幕范围校验（但非必须，UE4 通常会自动修正）
   - ScreenPosition 属性在 UE4.27 中可能通过
     .ScreenPosition(FVector2D) 设置，或通过
     SWindow->MoveWindowTo(FVector2D) 在创建后设置

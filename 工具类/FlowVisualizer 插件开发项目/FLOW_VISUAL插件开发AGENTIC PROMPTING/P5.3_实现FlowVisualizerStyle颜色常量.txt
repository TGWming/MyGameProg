【任务目标】
在 FlowVisualizerEditor 模块中创建 FlowVisualizerStyle 文件，
将设计规格中的全部配色方案、字体规格和动画常量集中定义为
命名空间常量和工具函数，供全插件引用，消除硬编码。

【具体要求】

1. ��要创建 2 个文件：

   文件 A：Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Private/FlowVisualizerStyle.h
   文件 B：Plugins/FlowVisualizer/Source/FlowVisualizerEditor/Private/FlowVisualizerStyle.cpp

2. 在 namespace FlowVisStyle 中定义以下颜色常量（均为 FLinearColor）：

   背景类：
   - Background      = FLinearColor(0.05f, 0.05f, 0.08f, 1.0f)   // #0D0D14
   - TitleBarBg      = FLinearColor(0.10f, 0.10f, 0.15f, 1.0f)   // #1A1A25

   节点类：
   - NodeActive      = FLinearColor(0.20f, 0.80f, 1.00f, 1.0f)   // #33CCFF
   - NodeInactive    = FLinearColor(0.40f, 0.40f, 0.40f, 1.0f)   // #666666
   - DataValue       = FLinearColor(0.29f, 1.00f, 0.51f, 1.0f)   // #4AFF82

   连线类：
   - EdgeDefault     = FLinearColor(0.53f, 0.53f, 0.53f, 1.0f)   // #888888
   - EdgeActive      = FLinearColor(0.20f, 0.80f, 1.00f, 1.0f)   // #33CCFF
   - FlowDot         = FLinearColor(1.00f, 1.00f, 1.00f, 1.0f)   // #FFFFFF

   UI 类：
   - Border          = FLinearColor(0.20f, 0.20f, 0.25f, 1.0f)   // #333340
   - StatusText      = FLinearColor(0.60f, 0.60f, 0.60f, 1.0f)   // #999999
   - Warning         = FLinearColor(1.00f, 0.40f, 0.27f, 1.0f)   // #FF6644

3. 定义 4 个字体获取函数（声明在 .h，实现在 .cpp）：

   FSlateFontInfo NodeTitleFont();
   - 返回 Roboto Bold 14pt
   - 实现：FCoreStyle::GetDefaultFontStyle("Bold", 14)

   FSlateFontInfo DataValueFont();
   - 返回 Roboto Mono 11pt
   - 实现：FCoreStyle::GetDefaultFontStyle("Mono", 11)

   FSlateFontInfo StatusBarFont();
   - 返回 Roboto Mono 10pt
   - 实现：FCoreStyle::GetDefaultFontStyle("Mono", 10)

   FSlateFontInfo PipelineTitleFont();
   - 返回 Roboto Bold 12pt
   - 实现：FCoreStyle::GetDefaultFontStyle("Bold", 12)

4. 定义动画/行为常量（均为 const float）：

   - HighlightFadeDuration = 1.5f
     （节点高亮渐灭持续时间，秒）
   - FlowDotSpeed = 200.0f
     （流动光点移动速度，像素/秒）
   - SignalThrottleMs = 100.0f
     （同节点信号采样节流窗口，毫秒）

5. 关键技术注意事项：
   - .h 中颜色和动画常量使用 extern const 声明，.cpp 中定义
     或者在 .h 中直接 inline / 命名空间内定义（C++14 兼容方式）
   - UE4.27 编译器支持 C++14，namespace 内 const 默认 internal linkage，
     直接在 .h 中定义即可，无需 extern
   - 字体函数需要在 .cpp 中实现，因为 FCoreStyle 调用不适合 inline
   - #include "Styling/CoreStyle.h" 获取 FCoreStyle
   - #include "Fonts/SlateFontInfo.h" 获取 FSlateFontInfo

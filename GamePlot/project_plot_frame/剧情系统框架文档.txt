# 《欺渊》  
**关卡剧情整合与合理化模板** v1.1  

**版本**：v1.1（融合另一个LLM关卡建议 + 我们v2.0叙事驱动系统）  
**核心目的**：让你把**已有关卡、美术、敌人分布、路线设计**快速对齐到剧情框架，确保每层都服务于“三轴数值 + 直觉被骗 vs 反直觉醒悟”的欺骗感。  
**使用方法**：  
1. 把你已有的关卡布局、美术描述、敌人波次、路线草图填入下方模板。  
2. 检查每层是否同时提供“直觉路径”和“反直觉路径”。  
3. 保存为本地文件，随时迭代。

---

## 1. 总体关卡哲学（必须遵守）
- 全游戏**单向下行**（无法返回），象征“越往下人性越被雨煮烂”。  
- 每层同时存在**直觉路径**（轻松、奖励高、英雄感 → Trust↑）和**反直觉路径**（绕路、牺牲、恶心感 → Awareness↑）。  
- 所有触发器均为**条件驱动**（非写死），根据三轴数值动态变化。  
- 美术/敌人/路线全部服务于**哀物凄厉美感**（纯生物血肉畸变体，无鬼魂）。

---

## 2. 触发器系统（新增通用模板）
**核心规则**：每个重要位置放置触发器，根据当前Trust / Corruption / Awareness动态改变台词、事件、奖励。  
**通用结构**（你可直接在引擎里实现）：
- 触发位置（Box / 交互点）
- 条件（Trust ≥ X / 是否阅读日记 / 是否击杀Boss等）
- 事件（修改数值、播放维兰台词、解锁/锁定路径、屏幕反馈）

## 3. 三层关卡模板（Layer 1 已按另一个LLM建议填充示例）

### 第1层：雨蚀地面（伪善日常的缓缓崩坏）
**美术风格**（建议）：永夜黑雨、倾斜木屋、纸窗血痕、灯笼摇曳肿胀人脸影子。  
**敌人分布**（可替换为你已有）：镇兵（直觉主力）+ 腐铠巨兵（反直觉隐藏）。  
**路线**：直觉主路（宽敞泥巷，按维兰指引） vs 反直觉支路（狭窄漏雨小巷）。

**关键触发器示例**（已融合另一个LLM的10个核心，供你直接套用/修改）：
| 序号 | 触发位置 | 条件 | 事件（数值变化 + 反馈） | 设计目的 |
|------|----------|------|-------------------------|----------|
| ① | 进入破教堂 | 无 | 播放维兰初见台词：“你……还活着……太好了……”<br>Trust +2<br>解锁武器修理 | 建立“唯一可靠的人”印象 |
| ② | 使用圣血弹 | 首次使用 | Trust +2 / Corruption +3<br>屏幕轻微血脉跳动 | 强化“直觉武器=好” |
| ③ | 任意镇兵死亡 | 敌人死亡 | Trust +1 | 建立“杀=正确”逻辑 |
| ④ | 阅读完整日记 | 完整阅读 | Trust -1 / Awareness +3 | 反直觉觉醒点 |
| ⑤ | 与维兰交互（交日记） | 主动上交 | Trust +3 / Awareness -2 | 玩家主动“放弃理解” |
| ⑥ | 走反直觉支路 | 未走主路径 | Trust -1 / Awareness +2 | 检测绕路行为 |
| ⑦ | 腐铠队长Boss战前 | Trust ≥ 5 | 维兰台词：“别犹豫……他已经不是人了……” | 强化依赖 |
| ⑧ | Boss击杀后 | 完成战斗 | Trust +3 / Corruption +2<br>掉落强力武器（但Corruption +5） | 奖励反转（表面强，隐藏代价高） |
| ⑨ | 返回教堂 | 回到维兰 | 免费强化 + 台词：“你一直在帮我……对吗？”<br>Trust +4 | 强化情感依赖 |
| ⑩ | 第一层出口 | Trust ≥ 8 | 正常进入下一层<br>Trust < 8 → 触发隐藏对话（维兰温柔挽留） | 出口锁定，增加“必须相信他”的压迫感 |

**Boss**：腐铠队长（死前低喃伦理碎片）。

### 第3层：血肉渊心（人性彻底被雨煮烂）
**美术风格**（建议）：肉壁如被雨打湿的旧纸、融合怪物仍有镇民轮廓。
